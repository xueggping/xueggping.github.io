GitHub配置发布

1.通过这条命令来新建日志

2.deploy配置

3.如果是windows 系统请安装 hexo-deployer-git

npm install hexo-deployer-git --save
如果下载太慢, 你可以使用我们定制的 cnpm (gzip 压缩支持) 命令行工具代替默认的 npm.成功后用cnpm 代替 npm 来安装 hexo-deployer-git 
$ npm install -g cnpm --registry=https://registry.npm.taobao.org

4.打包，部署

设置SSH keys

1.查看ssh值列表 如果有则删除所有的

ls -al ~/.ssh
2.设置ssh key 
ssh-keygen -t rsa -C "邮箱地址"
3.启动代理 
ssh-agent -s
ssh-add ~/.ssh/id_rsa 如果报错则执行： 
eval `ssh-agent -s`
ssh-add
拷贝key 
clip < ~/.ssh/id_rsa.pub
